# AI Powered Image Similarity, Search and Recommendation System

This project uses a pretrained deep learning model (ResNet50) to extract image features and find visually similar images using cosine similarity.

## Features
- Image similarity search
- Content-based image recommendation
- Deep learning feature extraction
- Streamlit web application

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Scikit-learn
- Streamlit

## How to Run
1. Install dependencies
2. Extract features
   ```bash
   python extract_features.py


3. Run
    streamlit run app.py
